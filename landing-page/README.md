# EthioTourism Landing Page

A modern, responsive landing page for the Ethiopian Tourism Information System built with Next.js 14 (App Router) and Tailwind CSS.

## Features

- **Hero Section**: Full-width hero with Ethiopian tourism imagery and CTAs
- **Navigation Bar**: Sticky navbar with mobile responsive hamburger menu
- **Featured Destinations**: Grid showcase of 6 iconic Ethiopian destinations
- **Tourism Categories**: Historical Sites, National Parks, and Cultural Festivals
- **Why Choose Ethiopia**: Feature cards highlighting unique aspects
- **Booking Promotion**: Call-to-action section for easy trip planning
- **Popular Festivals**: Showcase of major Ethiopian cultural celebrations
- **Testimonials**: Real traveler reviews and experiences
- **Call to Action**: Strong conversion section
- **Footer**: Comprehensive footer with links, contact info, and social media

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Lucide React (Icons)

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3001](http://localhost:3001) in your browser
